# Trading Translation Glossary

Use mainland Chinese trading-community wording. Prefer concise subtitle language over textbook phrasing.
- `first red day` / `the first red day`: `首阴日`。这是交易形态术语，不译成“第一个红日”或“首个亏损日”。

## Core Terms

- scalper: 剥头皮交易员 / 超短线交易员; never translate as 黄牛
- scalping: 剥头皮 / 超短线
- price action: 价格行为
- candlestick: K 线
- order flow: 订单流
- footprint chart / order-flow chart: 足迹图 / 订单流图
- Deep Charts: Deep Charts; keep product name in English unless the user asks otherwise
- smart money: 聪明钱
- informed money: 知情资金 / 信息优势资金
- retail trader: 散户交易者
- edge: 交易优势
- breakout: 突破
- pullback: 回调
- support / resistance: 支撑 / 阻力
- liquidity: 流动性
- absorption: 吸收
- aggressive buyers / sellers: 主动买盘 / 主动卖盘
- passive buyers / sellers: 被动买盘 / 被动卖盘
- delta: Delta; use 成交量 Delta when first introduced
- options Greeks / Greek letters: keep option Greeks in uppercase English: GAMMA, DELTA, VEGA, THETA. Do not translate them as 伽马、德尔塔、维加、西塔 in subtitles.
- cumulative volume delta: CVD / 累积成交量 Delta
- volume spread analysis: VSA / 量价分析
- failed auction: 失败拍卖
- fair value gap: FVG / 公平价值缺口
- value area: 价值区
- fixed profile: 固定成交量分布 / 固定区间成交量分布
- volume profile: 成交量分布
- imbalance: 失衡
- momentum phase: 动量阶段
- market structure: 市场结构
- change of character / CHoCH: CHoCH / 性格转变 / 市场结构转变

## Style Rules

- Translate for people who already watch trading videos; avoid academic finance wording.
- Keep terms like Delta, CVD, FVG, VSA when they are common on trading screens.
- Prefer “主动买盘/主动卖盘” over “激进买方/激进卖方” when talking about aggressive order flow.
- Prefer “吸收卖压/吸收买盘” over literal “吸收掉了” when the object is order-flow pressure.
- Avoid “业绩” for trading performance; use “交易表现” or “收益表现”.
- Avoid over-formal words such as “公允” when the trading community commonly says “公平价值缺口” or “FVG”.
